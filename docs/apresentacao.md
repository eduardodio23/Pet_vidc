# Roteiro de apresentação PetVida (5 a 10 minutos)

## 1. O que é o PetVida
- Sistema para gestão de clínica veterinária.
- Centraliza cadastro de pets, consultas, pagamentos e relatórios.
- Demonstra a integração entre banco de dados relacional e API REST.

## 2. Mostrar o DER
- Apresentar as entidades principais: veterinários, tutores, animais e consultas.
- Destacar o relacionamento entre tutor e animal, e entre animal e consulta.
- Explicar a importância das views e procedures para organização do sistema.

## 3. Demo ao vivo
- Mostrar o endpoint de listagem de veterinários.
- Exibir uma consulta agendada via procedure.
- Mostrar o log de trigger ou o comportamento do sistema ao concluir uma consulta.
- Demonstrar um relatório financeiro ou a lista de inadimplentes.

## 4. Dificuldades e aprendizados
- Ajustes de integração entre MySQL e Node.js.
- Trabalhar com procedures, views e regras de negócio em SQL.
- Aprendizado sobre organização de projeto, documentação e apresentação técnica.
